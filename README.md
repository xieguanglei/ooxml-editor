# OOXML Editor

一个纯前端的 OOXML 文件编辑器，可以预览和编辑 Office 文档（.docx, .xlsx, .pptx）的内部 XML 结构。

## 功能特性

- 📁 **文件上传**：支持拖拽或点击上传 OOXML 文件
- 🌲 **目录树浏览**：可折叠的目录树，展示文件内部结构
- ✏️ **XML 编辑**：带语法高亮和代码折叠的 XML 编辑器
- 🖼️ **图片预览**：支持 PNG、JPEG、GIF、WebP 格式
- 🔍 **搜索功能**：在 XML 内容中搜索
- 💾 **下载保存**：修改后重新打包下载

## 技术栈

- **框架**: React 18 + TypeScript
- **构建**: Vite
- **样式**: Tailwind CSS
- **编辑器**: CodeMirror 6
- **ZIP 处理**: JSZip

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 使用说明

1. 打开应用后，拖拽或点击上传一个 OOXML 文件
2. 在左侧目录树中浏览文件结构
3. 点击文件查看/编辑内容
4. 修改完成后点击"下载"按钮保存

## 注意事项

- 修改 XML 文件后无需手动保存，下载时会自动包含所有更改
- 不支持编辑的文件格式（如 .bin）会显示提示信息
- 破坏性修改可能导致 Office 无法打开文件
